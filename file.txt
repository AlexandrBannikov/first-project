 Прохожу курс по работе с Гитом на Яндекс Практикуме.

Git - система контроля версий, которая помогает отслеживать изменения в проекте. Это инструмент как для индивидуальной, так и для командной работы.

1.Чтобы Гит начал отслеживать изменения в проекте, папку с файлами этого проекта надо сделать Git-репозиторием((англ)repository - хранилище).

Для этого надо переместиться в эту папку и ввести команду git init ((англ)init - инициализировать).
cd desktop/first-project # перешли в нужную папку
git init # создали репозиторий.
Разгитить папку можно с помощью команд rm -rf .git # удалили папку git.
Где -r ключ(англ. recursive - рекурсивно) позволяет удалять папки вместе с их содержимым.
Где -f ключ(англ. forse - заставляет) избавит тебя от вопросов ты точно хочешь удалить эти файлы.

2.Проверить состояние репозитория - git status(англ. status - состояние)
