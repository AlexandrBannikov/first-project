Как исправить коммит.

1.Исправить только последний коммит можно при помощи команды --amend(англ. amend - исправить) 
  ВАЖНО! --amend работает только с последним коммитом HEAD! Для исправления других более ранних коммитов есть другие команды.
  Сначала добавляем не закоммиченный файл git add file.txt а потом команда amend
  Дополнить последний коммит новыми файлами git commit --amend --no-edit # добавили в последний коммит новые файлы.
  Опция --no-edit сообщает команде commit что нужно оставить последнее сообщение о коммите.

2.Изменить сообщение коммита можно с помощью команды git commit --amend -m 'Новое сообщение' 
  при этом меняется и хеш коммита.
