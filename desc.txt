1. Pwd принт ворд директори - покажи рабочую директорию, папка где ты находишься.
2. CD change directory сменить директорию или сменить папку где ты находишься.
3. CD ~ вернуться на рабочую директорию или на главную папку.
4. LS покажи содержимое папки, ls -a содержимое папки со скрытыми файлами.
5. CAT от (англ. concatenate and print - объеденить и распечатать) cat myfile.txt
6. Удаление файла команда rm file.txt удалили файл из текущей папки
7. Удалить папку можно командой rmdir(remove directory - удалить директорию) rmdir project # удалили директорию
    если конечно папка пуста. Если папка не пуста появиться сообщение что директория не пуста.Это защита от случайного удаления файлов.
    Удалить папку со всем содержимым можно с командой rm -r project # удалили папку со всеми файлами.
